Gestión de Archivos y Carpetas
Aprenderás a crear, renombrar y organizar archivos en carpetas.

¿Qué es un archivo?

Un archivo es cualquier elemento que guardamos en nuestra computadora. Puede ser un documento, una foto, un video o incluso una canción. Los archivos tienen un nombre y un tipo que los identifica, como "Documento.docx" o "Imagen.jpg". Son esenciales para almacenar y gestionar nuestra información.

Ejemplos de archivos:

Un documento de Word llamado "Tareas.docx".
Una imagen titulada "Vacaciones.jpg".
Una canción guardada como "Favorita.mp3".


¿Qué es una carpeta?

Las carpetas son como cajas virtuales que usamos para organizar y guardar nuestros archivos. Ayudan a mantener todo en orden y a encontrar más rápido lo que necesitamos. Por ejemplo, puedes tener una carpeta llamada "Fotos" donde guardes todas tus imágenes.

Características de una carpeta:

Puede contener archivos y otras carpetas (subcarpetas). Tiene un nombre que puedes personalizar. Facilita el acceso y la organización.

Cómo crear una carpeta

Haz clic derecho en un espacio vacío del escritorio o en cualquier ventana de tu computadora. Selecciona la opción Nuevo y luego haz clic en Carpeta. Escribe un nombre para la carpeta (por ejemplo, "Mis documentos") y presiona Enter. Consejo práctico: Usa nombres descriptivos para tus carpetas, como "Fotos de vacaciones" o "Proyectos de trabajo", para identificarlas fácilmente.

Cómo mover un archivo a una carpeta

Haz clic sobre el archivo que deseas mover y mantén presionado el botón izquierdo del mouse. Arrastra el archivo hacia la carpeta en la que deseas guardarlo. Suelta el botón del mouse para soltar el archivo dentro de la carpeta. Nota: Si prefieres otra opción, también puedes copiar el archivo (Ctrl+C) y pegarlo (Ctrl+V) dentro de la carpeta.

Actividad práctica

Actividad práctica
Manos a la obra:

Crea una carpeta en el escritorio llamada "Mis documentos". Busca un archivo en tu computadora y guárdalo dentro de esa carpeta. Renombra la carpeta a "Mis documentos personales" para practicar cómo cambiar el nombre de una carpeta. Recomendaciones para organizar tus carpetas y archivos:


Usa categorías claras: Organiza tus carpetas por temas o proyectos. Evita el desorden: No guardes todos tus archivos en el escritorio. Haz respaldos: Guarda copias de tus archivos importantes en una memoria USB o en la nube. Ahora estás listo para gestionar tus archivos y carpetas como un experto. ¡Sigue practicando para dominar estas herramientas esenciales!
