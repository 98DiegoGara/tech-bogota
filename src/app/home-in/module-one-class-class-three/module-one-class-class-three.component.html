Navegación en Internet
Explora el mundo desde tu computadora: Aprende a usar un navegador para buscar información y navegar de manera segura. En esta lección, te enseñaremos qué es internet, cómo abrir un navegador y realizar búsquedas efectivas, además de consejos para identificar información confiable.

¿Qué es internet?

Internet es una gigantesca red que conecta computadoras y dispositivos de todo el mundo. Nos permite: Acceder a información sobre cualquier tema. Ver videos, escuchar música y disfrutar de entretenimiento. Comunicarnos con personas mediante correos electrónicos, chats o videollamadas.
Ejemplo práctico: Gracias a internet, puedes aprender nuevas recetas, explorar destinos turísticos y resolver dudas con un par de clics.

¿Qué es un navegador?

Un navegador es un programa que sirve como puerta de entrada a internet. Es el que usamos para visitar páginas web, buscar información o ver videos.
Ejemplos de navegadores populares:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (en dispositivos Apple)
  • Dato útil: Aunque todos cumplen funciones similares, algunos navegadores pueden ser más rápidos o tener características específicas.

    •  Cómo abrir un navegador

    Localiza el ícono del navegador en el escritorio o barra de herramientas de tu computadora. Ejemplo: El ícono de Google Chrome tiene forma de círculo con colores rojo, verde, amarillo y azul.
    Haz doble clic en el ícono para abrir el navegador. Consejo: Si no encuentras el navegador, usa la barra de búsqueda de tu computadora para escribir el nombre del programa, como "Chrome" o "Firefox".

    Cómo buscar algo en internet

    Identifica la barra de búsqueda: Está en la parte superior del navegador. Escribe lo que deseas buscar. Por ejemplo: "clima en Bogotá" o "recetas fáciles". Presiona Enter para ver los resultados. Haz clic en un enlace de la lista para abrirlo y explorar la información. Tip para búsquedas efectivas: Usa palabras clave claras y específicas para obtener mejores resultados.

    Cuidado con la información en internet

    Prefiere sitios conocidos: Busca información en páginas oficiales, educativas o de instituciones reconocidas. Verifica varias fuentes: Si algo parece dudoso, compáralo con otros sitios. Evita hacer clic en anuncios sospechosos: Algunos enlaces pueden contener virus o información falsa.
